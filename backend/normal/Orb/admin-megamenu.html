About MegaMenuInfo

Please note that ORB MegaMenu Required JavaScript to operate. It also uses Gmap plugin for pleasant and comfortable work with Google Maps. See documentation for more details.
There are so my to orginize content and navigation inside dashboard. We have already make an example of default Bootstrap top navbar implementation. To give you more flexibility we include our alternative navigation panel based on Bootstrap navbar. ORB Responsive MegaMenu can contains variety of elements, such as responsive images, responsive videos, responsive Google Maps and much more. We are also include 6 predifined ordered and unordered lists styles. Megamenu can be used as a default top navigation panel for ORB.
