WYSIWYG Bootstrap Usage

This is a tiny (5KB, < 200 lines) jQuery Bootstrap plugin turns any DIV into a WYSIWYG rich-content editor, inspired by CLEditor and bootstrap-wysihtml5. The documentation for WYSIWYG Bootstrap is available on their website, mindmup.github.io/bootstrap-wysiwyg/.
