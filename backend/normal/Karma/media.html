Responsive jQuery lightbox plugin that is focused on performance and providing best experience for user with any device. This plugin can play video's.

You can use a lightbox to present a bigger image.

Usage: .lightbox
Gallery

You can group image to form a gallery. Notice: the class name must be unique to form different gallerys on the same page.

Usage: .lightbox-group-1

Iframe
