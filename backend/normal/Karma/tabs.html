Bootstrap responsive tabs with AJAX

This custom made plugin will extend the bootstrap tabs plugin with AJAX and a responsive mode. The AJAX files can be preloaded or can be loaded if the tab gets selected. If the tabs goes into responsive mode it will transform into a simple accordion. The AJAX part can also work with ext-tabs-btn-group tabs.
